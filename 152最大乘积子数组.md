给你一个整数数组 `nums` ，请你找出数组中乘积最大的非空连续 子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 **32-位** 整数。

**请注意**，一个只包含一个元素的数组的乘积是这个元素的值。

**示例 1:**

**输入:** nums = `[2,3,-2,4]`
**输出:** `6`
**解释:** 子数组 `[2,3]` 有最大乘积 6。

**示例 2:**

	**输入:** nums = `[-2,0,-1]`
**输出:** 0
**解释:** 结果不能为 2, 因为 `[-2,-1]` 不是子数组。

**提示:**

- `1 <= nums.length <= 2 * 104`
- `-10 <= nums[i] <= 10`
- `nums` 的任何子数组的乘积都 **保证** 是一个 **32-位** 整数

很显然可以拆分成子问题，对于下标为i的元素来说，如果希望以第i个元素为右端点的连续子数组的乘积最大，那么希望以第i-1个元素为右端点的连续子数组的乘积最小/最大。如果第元素是负数，那么就希望是最小，如果是正数，就希望是最大

用`dp[i]`表示以第i个元素为右端点的连续子数组的乘积，`a[i]`表示数组下标为i的元素，可以得到：

若`a[i] < 0`，则`dp[i]_Max = dp[i]_Min * a[i]`
若`a[i] > 0`，则`dp[i]_Max = dp[i]_Max * a[i]`
若`a[i] = 0`，那么`dp[i]_Max = 0`，但难道`dp[i+1]`也会为0吗？显然不一定，它可以自己作为子数组

因此可以想到，还有一种情况是，压根不需要考虑`dp[i-1]`，例如`dp[i-1]`是非正数而`a[i] > 0`，这种时候`a[i]`应该自己独立作为子数组

所以可得：

`dp[i]_Max = Max(dp[i-1]_Max * a[i], dp[i-1]_Min * a[i], a[i]`

注意到3件事：
1.状态上`dp[i]`只和`dp[i-1]`有关，所以不需要开辟数组，使用变量做滚动数组即可
2.必须要维护`dp[i-1]_Max`和`dp[i-1]_Min`，用两个变量dpMax和dpMin记录
3.如果`a[i] < 0`，那么左Max右Min（等式左边是`dp[i]_Max`右边是`dp[i]_Min`），反之左Max右Max，因此计算前可以加一个if分支，如果`a[i] < 0`，那么就交换dpMax和dpMin，这样后续的代码就统一是左Max右Max，左Min右Min

需要注意，如果使用滚动数组，就需要做好记忆化，因为如果你先算dpMin，再算dpMax，由于dpMin已经更新了，但dpMax要用的dpMin是旧的dpMin，所以结果就出问题了。因为这个时候状态转移方程不是`dp[i]_Max = Max(dp[i-1]_Max * a[i], dp[i-1]_Min * a[i], a[i]`，而是`dp[i]_Max = Max(dp[i-1]_Max * a[i], dp[i]_Min * a[i], a[i]`

但是我们加了`if(a[i] < 0)`的分支，作了交换，统一左Max右Max，左Min右Min，Max不用到Min，Min不用到Max，就不会出现想用旧值但实际是新值的问题

```java
class Solution {
    public int maxProduct(int[] nums) {
        int ans = Integer.MIN_VALUE;
        int dpMax = 1; //初始值为1，方便做乘法
        int dpMin = 1;
        for(int a : nums) {
            //dpMin = Min(dpMin*a, dpMax*a, a)
            //dpMax = Max(dpMin*a, dpMax*a, a)
            if(a < 0) {
                //如果a小于0，那么原本的最小值dpMin乘a后就会变成最大值，最大值dpMax乘a后就会变成最小值
                //因此，如果要求dpMin, 那么由于dpMax*a·必然小于dpMin*a, 因此可以去掉dpMin*a的比较。求dpMax同理
                //交换dpMin和dpMax
                //因为如果a大于0，那么求dpMin时dpMin乘a后必然小于dpMax乘a
                //而且交换的同时也相当于做了记忆化，保留了当前的dpMin和dpMax
                int tmp = dpMin;
                dpMin = dpMax;
                dpMax = tmp;
            }
            dpMin = Math.min(dpMin*a, a);
            dpMax = Math.max(dpMax*a, a);
            ans = Math.max(ans, dpMax);
        }
        return ans;
    }
}
```
