给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums` ，**[原地](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** 对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 `0`、 `1` 和 `2` 分别表示红色、白色和蓝色。

必须在不使用库内置的 sort 函数的情况下解决这个问题。

**示例 1：**

**输入：nums = `[2,0,2,1,1,0]`
输出：`[0,0,1,1,2,2]`

**示例 2：**

**输入：nums = `[2,0,1]`
输出：`[0,1,2]`

**提示：**

- `n == nums.length`
- `1 <= n <= 300`
- `nums[i]` 为 `0`、`1` 或 `2`

**进阶：**

- 你能想出一个仅使用常数空间的一趟扫描算法吗？

### 方法一：单指针，两趟

第一趟先把0往前扔，第二趟把1放到0后面

用一个指针维护已排列部分的队尾即可知道下一个需要排列的位置在哪

Java

```java
class Solution {
    public void sortColors(int[] nums) {
        //第一轮把0放前面，第二轮把1放0后面
        //用一个单指针标记已排列部分的末尾（已排列部分最后一个元素的下一个元素），一开始指针在0号元素
        int n = nums.length;
        int ptr = 0;
        //第一轮
        for(int i=0;i<n;i++) {
            if(nums[i] == 0) {
                int temp = nums[i];
                nums[i] = nums[ptr];
                nums[ptr] = temp;
                ptr++;
            }
        }

        //第二轮
        for(int i=ptr;i<n;i++) {
            if(nums[i] == 1) {
                int temp = nums[i];
                nums[i] = nums[ptr];
                nums[ptr] = temp;
                ptr++;
            }
        }
    }
}
```

Python

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n = len(nums)
        ptr = 0
        for i in range(n) :
            if nums[i] == 0 :
                nums[i],nums[ptr] = nums[ptr],nums[i]
                ptr+=1
        for i in range(ptr,n):
            if nums[i] == 1 :
                nums[i],nums[ptr] = nums[ptr],nums[i]
                ptr+=1

```

	ptr := 0
	for i := range nums {
		if nums[i] == 0 {

### 方法二：计数，一趟

一边处理数组，一边记录0和1的个数

先记录元素的值，然后将该位置赋值为2，因为如果不是2的话需要计数，所以统一先赋值为2

引入两个指针，p0记录已赋值为0的序列的最后一个位置，p1记录已赋值为1的序列的最后一个位置

如果当前元素是0或1，那么就先将其赋值为1，并且将p1后移。因为不管是0还是1，0一定是放在1前面的，所以已赋值为1的序列的最后一个位置必须要后移

进一步看当前元素是否为0，如果为0，p0也要后移，由于此时p1已经后移，所以不需要再移动

实际上你可以视为我们又新开辟了一个数组用来填0 1 2，因此只需要知道原数组中0和1的个数即可，p0实际上就是已遍历数组中0的数量，p1-p0就是1的数量

```java
class Solution {
    public void sortColors(int[] nums) {
        int p0 = 0;
        int p1 = 0;
        int n = nums.length;
        for(int i=0;i<n;i++) {
            int temp = nums[i];
            nums[i] = 2;
            if(temp <= 1) {
                nums[p1++] = 1;
            }
            if(temp == 0) {
                nums[p0++] = 0;
            }
        }
    }
}
```
