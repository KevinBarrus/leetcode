给定一个 _n_ × _n_ 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在 **[原地](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** 旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要** 使用另一个矩阵来旋转图像。

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/08/28/mat1.jpg)

**输入：**matrix = `[[1,2,3],[4,5,6],[7,8,9]]`
**输出：**`[[7,4,1],[8,5,2],[9,6,3]]`

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/08/28/mat2.jpg)

**输入：**matrix = `[[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]`
**输出：**`[[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]`

**提示：**

- `n == matrix.length == matrix[i].length`
- `1 <= n <= 20`
- `-1000 <= matrix[i][j] <= 1000`

相当于人从正面看，变成人从右边看

假设原先矩阵有 n 行，n 列

原先的第0列，旋转后变为第0行；原先的第1列，旋转后变为第1行；原先的第 j 列，旋转后变为第 j 行

原先的第0行，旋转后变为第 n - 1 列；原先的第1行，旋转后变为第 n - 2 列；原先的第 i 行，旋转后变为第 n - 1 - i 列

所以，原先的 (i, j) ，旋转后变为 (j, n - 1 - i)

但是，不能直接 swap，因为必然要 swap 两次，但 swap 两次就相当于没有交换，实际上是因为直接交换缺少“对称性”，如果有对称性，那么只需要遍历数组的一半，然后只需执行一次 swap 

(i, j) -> (j, n - 1 - i) 可以视为：
(i, j) -> (j, i) -> (j, n - 1 - i)

也就是先对矩阵进行转置，再沿着竖直中轴线翻转

而转置本质上是沿主对角线翻转，因此也具有对称性。所以这两步操作都具有对称性，可以实现

Java

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        for (int i = 0; i < n; i++) {
            int[] row = matrix[i];
            // 遍历对角线上方元素，做 (i, j) -> (j, i) 交换，即转置
            for(int j = i + 1; j < n; j++) {
                int tmp = row[j];
                row[j] = matrix[j][i];
                matrix[j][i] = tmp;
            }

            // 遍历左半元素，做 (j, i) -> (j, n - 1 - i) 交换，即沿着竖直中轴线翻转
            for(int j = 0; j < n / 2; j++) {
                int tmp = row[j];
                row[j] = row[n - 1 - j];
                row[n - 1 - j] = tmp;
            }
        }
    }
}
```

注意这里必须要先遍历上三角而不是下三角，以示例一为例

如果是遍历下三角：
`[1, 2, 3]` 无转置元素，沿中轴线翻折后变成 `[3, 2, 1]`
`[4, 5, 6]` 中4和2对调，沿中轴线翻折后变成 `[6, 5, 2]`
`[7, 8, 9]` 中7和1对调，8和2对调，沿中轴线翻折后变成 `[9, 2, 1]`

最终变为：
```
[3, 4, 7]
[6, 5, 8]
[9, 2, 1]
```

下三角是一开始就先列变换再转置，顺序错了

上三角能保证每次都是先转置再列变换

Python

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        n = len(matrix)
        for i, row in enumerate(matrix) :
            for j in range(i + 1, n) :
                row[j], matrix[j][i] = matrix[j][i], row[j]
            row.reverse() # 行翻转
```
